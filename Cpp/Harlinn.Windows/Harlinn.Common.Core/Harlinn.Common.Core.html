Harlinn.Common.Core

The Harlinn.Common.Core C++ library contains C++ classes that enables safer C++ application development. It has been used to create critical, high performance servers, that stores, retrieves, and analyzes data belonging to more than a million timeseries.

PBRTO a micro optimized raytracing app

PBRTO is a micro optimized version of PBRT-v4, created as an example of how the functionality in HCCMath.h and HCCSIMD.h can be used to optimize the performance of a real, computationally intensive, app. It’s now about 35 % faster than the release build of the original PBRT. more…

Optimized basic math functions

C++ is often used to create software that performs mathematical operations efficiently.

Complex computations are put together using the functionality provided by the standard C and C++ libraries as the basic building blocks. The header, HCCMath.h provides alternatives to many of the basic mathematical function that is specified in the C and C++ standards. All of the functions can be constexpr evaluated, and several offer surprising performance benefits as well. more…
