Adds two uint16 vectors.

Parameters
lhsuint16 vector used for the left-hand side of the operation
rhsuint16 vector used for the right-hand side of the operation

Adds packed unsigned 16-bit integers from source vector lhs and corresponding bits of source vector rhs and stores the packed integer result in the destination vector. When an individual result is too large to be represented in 16 bits (overflow), the result is wrapped around and the low 16 bits are written to the destination vector (that is, the carry is ignored). You must control the range of values operated upon to prevent undetected overflow conditions.

Returns
Result of the addition operation.

Subtracts uint16 vectors.

Parameters
lhsuint16 vector used for the left-hand side of the operation
rhsuint16 vector used for the right-hand side of the operation

Subtracts packed unsigned 16-bit integers of the second source vector rhs from corresponding bits of the first source vector lhs. When an individual result is too large to be represented in 16 bits (overflow), the result is wrapped around and the low 16 bits are written to the destination vector (that is, the carry is ignored).

You must control the range of values operated upon to prevent undetected overflow conditions.

Returns
Returns the result of the subtraction operation.
