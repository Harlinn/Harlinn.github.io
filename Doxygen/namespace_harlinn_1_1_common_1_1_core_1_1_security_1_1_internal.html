enum class Harlinn::Common::Core::Security::Internal::ReleaseMethod
strong

Indicates how resources are released when the managing object goes out of scope.

Enumerator
None 

No action is taken to release the resource.

LocalFree 

LocalFree is called to release the memory for the resource.

FreeSid 

The resource is a SID and FreeSid is called to relese the memory for the SID.

Delete 

The memory was allocated using <emp>new Byte[size]</emp> and is released using delete.

DestroyPrivateObjectSecurity 

The memory will be released using DestroyPrivateObjectSecurity.
