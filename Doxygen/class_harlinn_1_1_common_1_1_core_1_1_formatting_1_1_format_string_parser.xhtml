Detailed Description

template<typename CharT, size_t expectedSectionCount = 10, size_t workBufferSize = 256>
class Harlinn::Common::Core::Formatting::FormatStringParser< CharT, expectedSectionCount, workBufferSize >

Splits a format string into sections, a section will either hold text to be copied to the output, or a format specification.

Template Parameters
CharTThe character type
expectedSectionCountThe number of sections to reserve space for
workBufferSizeThe number of characters to reserve spece for in the work area/buffer
