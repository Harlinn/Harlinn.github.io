Detailed Description

Optionally supported by media sinks to perform required tasks before shutdown. This functionality is typically exposed by archive sinks—that is, media sinks that write to a file. It is used to perform tasks such as flushing data to disk or updating a file header.

To get an instance of this class, call As<MFFinalizableMediaSink>() on the media sink.
