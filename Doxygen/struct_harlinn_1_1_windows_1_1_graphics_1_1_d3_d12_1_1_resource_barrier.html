Creates a ResourceBarrier that describes the transition of subresources between different usages.

Parameters
resourceThe resource used in the transition.
stateBeforeThe "before" usages of the subresources, as a bitwise-OR'd combination of ResourceStates enumeration constants.
stateAfterThe "after" usages of the subresources, as a bitwise-OR'd combination of ResourceStates enumeration constants.
flagsSpecifies a value from the ResourceBarrierFlags enumeration.
subresourceThe index of the subresource for the transition. Defaults to the D3D12_RESOURCE_BARRIER_ALL_SUBRESOURCES flag ( 0xffffffff ) which transitions all subresources in a resource at the same time.

Creates a ResourceBarrier that describes the transition between usages of two different resources that have mappings into the same heap.

Parameters
resourceBeforeRepresents the before resource used in the transition.
resourceAfterRepresents the after resource used in the transition.
flagsSpecifies a value from the ResourceBarrierFlags enumeration.

Both the before and the after resources can be specified or one or both resources can be NULL, which indicates that any placed or reserved resource could cause aliasing.
