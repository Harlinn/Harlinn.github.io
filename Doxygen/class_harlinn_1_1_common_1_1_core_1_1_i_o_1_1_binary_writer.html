Detailed Description

template<IO::StreamWriter StreamT, bool networkByteOrder = false, bool use7BitEncodedSize = true>
class Harlinn::Common::Core::IO::BinaryWriter< StreamT, networkByteOrder, use7BitEncodedSize >

This template is used to implement serialization of binary data to stream objects that satisfies the IO::StreamWriter concept.

Template Parameters
StreamTA type that satisfies the IO::StreamWriter concept.
networkByteOrderIf true the template will convert data to network byte order from little endian format, otherwise false.
use7BitEncodedSizeIf true size information will written as 7-bit encoded unsigned numbers, otherwise false.
