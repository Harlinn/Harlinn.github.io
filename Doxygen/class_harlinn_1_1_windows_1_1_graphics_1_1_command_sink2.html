Detailed Description

This interface performs all the same functions as the existing ID2D1CommandSink1 interface. It also enables access to ink rendering and gradient mesh rendering.

Draws a metafile to the command sink using the given source and destination rectangles.

Parameters
gdiMetafileThe metafile to draw.
destinationRectangleThe rectangle in the target where the metafile will be drawn, relative to the upper left corner (defined in DIPs). If nullptr is specified, the destination rectangle is the size of the target.
sourceRectangleThe rectangle of the source metafile that will be drawn, relative to the upper left corner (defined in DIPs). If nullptr is specified, the source rectangle is the value returned by ID2D1GdiMetafile1::GetSourceBounds.
