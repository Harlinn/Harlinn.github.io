Detailed Description

template<typename Derived, bool networkByteOrder, bool use7BitEncodedSize>
class Harlinn::Common::Core::IO::BinaryReaderBase< Derived, networkByteOrder, use7BitEncodedSize >

This template is used to implement deserialization of binary data using a Read function implemented by the derived class with the following signature:

long long Read( void* buffer, size_t bufferSize );

Template Parameters
DerivedThe type of the derived class.
networkByteOrderIf true the template will convert data from network byte order to little endian format, otherwise false.
use7BitEncodedSizeIf true size information is expected to be 7-bit encoded, otherwise false.

This class supports reading of the following types from the underlying stream:

  • bool
  • char
  • SByte
  • Byte
  • wchar_t
  • Int16
  • UInt16
  • Int32
  • long
  • UInt32
  • unsigned long
  • Int64
  • UInt64
  • float
  • double
  • TimeSpan
  • DateTime
  • Currency
  • Guid
  • std::string
  • std::wstring
  • AnsiString
  • WideString
  • Binary
  • enum types
  • std::optional<T> containing any of the above.

The template can also read length prefixed data from the underlying stream into std::vector, std::array and c-style arrays, where the value type is any of the above.
