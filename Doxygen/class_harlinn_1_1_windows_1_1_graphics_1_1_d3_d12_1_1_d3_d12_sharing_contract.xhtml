Detailed Description

Facilitates diagnostics tools to capture information at a lower level than the DXGI swapchain.

You may want to use this class to enable diagnostic tools to capture usage patterns that don't use DXGI swap chains for presentation. If so, you can create and instance of this class by calling the static Create function passing the queue as its parameter.

Note that the underlying interface is not supported when there are no diagnostic tools present, so your application mustn't rely on it existing.
