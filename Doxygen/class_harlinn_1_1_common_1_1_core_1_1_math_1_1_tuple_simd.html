Detailed Description

template<typename TraitsT, typename TupleT>
class Harlinn::Common::Core::Math::TupleSimd< TraitsT, TupleT >

Holds the SIMD value for a Tuple2, Tuple3 or Tuple4.

Mathematical operations for a particular Tuple2, Tuple3 or Tuple4 derived type should generally be performed as a sequence of operations on this type, not the Tuple2, Tuple3 or Tuple4 derived type, as this type represents the loaded state of the data.

Template Parameters
TraitsTThe SIMD::Traits specialization that will be used to select the SIMD intrinsics that operates on the simd value held by this TupleSimd.
TupleTA Tuple2, Tuple3 or Tuple4 derived type.

This implementation passes TupleSimd objects by reference, since passing the TupleSimd objects by value degraded the performance of pbrto by approximately 10 %.
