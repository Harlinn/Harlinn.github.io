Detailed Description

template<IO::StreamReader StreamT, bool networkByteOrder = false, bool use7BitEncodedSize = true>
class Harlinn::Common::Core::IO::BinaryReader< StreamT, networkByteOrder, use7BitEncodedSize >

This template is used to implement deserialization of binary data from stream objects that satisfies the IO::StreamReader concept.

Template Parameters
StreamTA type that satisfies the IO::StreamReader concept.
networkByteOrderIf true the template will convert data from network byte order to little endian format, otherwise false.
use7BitEncodedSizeIf true size information is expected to be 7-bit encoded, otherwise false.
