Detailed Description

RecordStream implements: Int64 Write( const void* buffer, size_t bufferSize ); which is required by IO::BinaryWriter<> in ThreadWriter::WriteLogRecord.

Instances of this class should not need any synchronization as they are supposed to only be used by the ThreadWriter class.
