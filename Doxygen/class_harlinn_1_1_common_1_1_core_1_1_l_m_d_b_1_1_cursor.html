void Harlinn::Common::Core::LMDB::Cursor::Delete ( bool noDupData = false)
inline

Delete current key/data pair.

This function deletes the key/data pair to which the cursor refers. This does not invalidate the cursor, so operations such as MoveNext() can still be used on it.

Parameters
noDupDatadelete all of the data items for the current key. This flag may only be specified if the database was opened with DatabaseFlags::DupSort
