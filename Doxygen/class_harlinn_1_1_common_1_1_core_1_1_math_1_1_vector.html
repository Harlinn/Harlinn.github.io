Detailed Description

template<typename ValueT, size_t N>
class Harlinn::Common::Core::Math::Vector< ValueT, N >

Vector is specialized for:

  • Vector<float,2>
  • Vector<float,3>
  • Vector<float,4>

The specializations are crafted for use in graphic apps.
