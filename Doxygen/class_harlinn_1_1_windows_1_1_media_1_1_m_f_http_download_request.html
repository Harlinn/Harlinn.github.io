Detailed Description

Applications implement this interface to override the default implementation of the HTTP and HTTPS protocols used by Microsoft Media Foundation. Applications provide the IMFHttpDownloadRequest interface to Media Foundation through the CreateRequest method on the MFHttpDownloadSession interface.
