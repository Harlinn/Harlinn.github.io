inlinestatic

Creates a system-memory buffer object to hold 2D image data.

Parameters
widthWidth of the image, in pixels.
heightHeight of the image, in pixels.
fourCCA FOURCC code or D3DFORMAT value that specifies the video format. If you have a video subtype GUID, you can use the first DWORD of the subtype.
bottomUpf true, the buffer's MF2DBuffer::ContiguousCopyTo method copies the buffer into a bottom-up format. The bottom-up format is compatible with GDI for uncompressed RGB images. If this parameter is false, the ContiguousCopyTo method copies the buffer into a top-down format, which is compatible with DirectX.
Returns
The new MFMediaBuffer object

static MFMediaBuffer Harlinn::Windows::MFMediaBuffer::CreateMediaBufferWrapper ( IMFMediaBuffer * buffer,
DWORD offset,
DWORD length )
inlinestatic

Creates a media buffer that wraps an existing media buffer. The new media buffer points to the same memory as the original media buffer, or to an offset from the start of the memory.

Parameters
bufferA pointer to the IMFMediaBuffer interface of the original media buffer.
offsetThe start of the new buffer, as an offset in bytes from the start of the original buffer.
lengthThe size of the new buffer. The value of offset + length must be less than or equal to the size of valid data the original buffer. (The size of the valid data is returned by the MFMediaBuffer::GetCurrentLength method.)
Returns
The new MFMediaBuffer object
