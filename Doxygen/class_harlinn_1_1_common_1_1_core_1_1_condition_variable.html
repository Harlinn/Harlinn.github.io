Sleeps on the specified condition variable and releases the specified critical section as an atomic operation.

Parameters
criticalSectionThe critical section object. This critical section must be entered exactly once by the caller at the time of the call.
timeoutInMillisThe time-out interval, in milliseconds. If the time-out interval elapses, the function re-acquires the critical section and returns zero. If dwMilliseconds is zero, the function tests the states of the specified objects and returns immediately. If dwMilliseconds is INFINITE, the function's time-out interval never elapses.
Returns
If the function succeeds, the return value is true. If function returns false to indicate that the time-out interval has elapsed before another thread has attempted to wake the sleeping thread.
