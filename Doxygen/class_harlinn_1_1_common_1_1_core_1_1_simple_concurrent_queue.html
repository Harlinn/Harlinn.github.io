Detailed Description

template<typename T, size_t maxQueueSize>
class Harlinn::Common::Core::SimpleConcurrentQueue< T, maxQueueSize >

Simple blocking fixed size queue.

Pop will return false when the queue is empty and closed. Push will return false when the queue is closed. If the producer was waiting for the queue, the item will not be placed in the queue.

Template Parameters
TThe type of elements to put in the queue
