Detailed Description

template<typename BaseT>
class Harlinn::Common::Core::Com::HeapObject< BaseT >

Use this template to instantiate an object with an implementation of IUnknown::Release() that calls the delete operator on itself when its reference count becomes 0. COM objects instantiated using this template must be allocated using the new operator.

Template Parameters
BaseTA type that implements one or more COM interfaces, but not IUnknown.
