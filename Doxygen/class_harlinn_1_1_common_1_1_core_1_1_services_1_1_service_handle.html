void Harlinn::Common::Core::Services::ServiceHandle::StartService ( DWORD serviceArgumentCount = 0,
LPCWSTR * serviceArguments = nullptr ) const
inline

Starts the service.

Parameters
serviceArgumentCountThe number of strings in the serviceArguments array. If serviceArguments is NULL, this parameter can be zero.
serviceArgumentsThe null-terminated strings to be passed to the ServiceMain function for the service as arguments. If there are no arguments, this parameter can be NULL. Otherwise, the first argument (serviceArguments[0]) is the name of the service, followed by any additional arguments (serviceArguments[1] through serviceArguments[serviceArgumentCount-1]).
