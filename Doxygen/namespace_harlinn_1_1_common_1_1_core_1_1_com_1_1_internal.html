Detailed Description

Use this template to implement COM objects that will be instantiated using either template<typename BaseT> class HeapObject {...} or template<typename BaseT> class StackObject {...}

HeapObject and StackObject provides the implementation of the IUnknown interface, while the remaining functions specified by the interfaces given in the InterfaceTypes parameter pack must be implemented by the class derived from ObjectBase.
