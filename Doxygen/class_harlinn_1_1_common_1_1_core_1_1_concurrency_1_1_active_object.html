Detailed Description

template<typename MessageT, size_t maxQueueSize>
class Harlinn::Common::Core::Concurrency::ActiveObject< MessageT, maxQueueSize >

Extends the ActiveObjectBase template, implementing the message processing thread and the Start and Stop functions.

Template Parameters
MessageTThe message type for the active object.
maxQueueSizeThe maximum queue size for the active object.
