Detailed Description

template<typename SemaphoreT = Semaphore, Int32 maxSpins = 10000>
class Harlinn::Common::Core::Internal::LightweightSemaphoreImpl< SemaphoreT, maxSpins >

LightweightSemaphoreImpl.

This is a modified version of Cameron Desrochers LightweightSemaphore.

I have changed it to work with my Semaphore class, and converted it into a template that takes an Int32 argument specifying the maximum spin count. Timeout are specified in milliseconds.

Original file comment:
Provides an efficient implementation of a semaphore (LightweightSemaphore).
This is an extension of Jeff Preshing's sempahore implementation (licensed
under the terms of its separate zlib license) that has been adapted and
extended by Cameron Desrochers.
